# 股票金融因子模型（Fama-French 三因子扩展研究）

这是一个围绕 Fama-French 三因子模型展开的研究型项目，目标是对股票超额收益进行建模，并探索在时变参数、损失函数与 GAM（广义可加模型）方向上的扩展与改进。

## 项目结构

- `Python代码示例/`：核心示例代码（数据读取、特征分析、建模与可视化）
- `备用代码/`：不同实现方案与备选版本
- `数据/`：示例数据与预测输出
- `研究+方案/`：研究思路、方案文档与材料

## 研究内容（简要）

- 基于 Fama-French 因子解释股票收益
- 引入时变建模思路
- 尝试不同损失函数以提升稳健性
- 使用 GAM 进行非线性关系刻画

## 运行说明

1. 准备 Python 3.9+ 环境
2. 安装常见依赖（如 `pandas`、`numpy`、`statsmodels`、`matplotlib`）
3. 按 `Python代码示例/` 中脚本顺序运行：
   - `数据读取与预处理.py`
   - `数据特征分析.py`
   - `时变方案.py` / `损失函数方案.py` / `GAM方案.py`
   - `统计指标与可视化.py`
